Color Name Prediction

Welcome to my Color Name Prediction Application

How this application works?
This app takes an image from the user. When user clicks on any part of the image, The rgb value of the pixel is stored and given to the pre trained Machine Learning Model. The model will take rgb value and predicts the color name. The final color name is displyed to the user.
How to use this app?
  • Select an image.
    • If the image is from internet copy the image address and paste it into the text box below.
      (NOTE: Some images may not be accessible)
    • If the image is from you device. Click on Choose file button and select the image from your device.
  • The image will be shown below.
  • Click on any part of the image to get the color name.
  • Color name is shown above the image.
